# Galería Interactiva con JavaScript

Proyecto universitario que implementa una galería interactiva de imágenes de motos utilizando JavaScript y manipulación del DOM.

## Funcionalidades
- Carga de imágenes locales por defecto
- Agregar imágenes mediante URL
- Selección de una imagen a la vez
- Eliminación de imágenes
- Atajo de teclado (Delete)

## Tecnologías Utilizadas
- HTML
- CSS
- JavaScript
